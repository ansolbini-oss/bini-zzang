# `/bini-zzang:기획-시장조사` — Phase 1 단독 실행

Phase 1(시장조사 + 전략)만 수행한다. Phase 0(용어집)이 없으면 먼저 생성, 있으면 재사용.

## 실행 단계

1. `.plan/00_glossary.md` 존재 확인
   - 없으면: `glossary-writer` 먼저 호출
   - 있으면: 재사용

2. `market-researcher` 호출
   - 입력: 주제, 도메인, 사용자 자료 경로
   - 산출물: `.plan/01_market.md` + `.plan/_constants.json`
   - Mid-gate: 페르소나 시드·페인 검토

3. `strategist` 호출
   - 입력: 01_market.md, 00_glossary.md
   - 포지셔닝 3안 → 사용자 선택
   - 산출물: `.plan/02_strategy.md`
   - Post-gate: Non-Goals 확정
